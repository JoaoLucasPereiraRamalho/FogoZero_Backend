```
  "status": "online"
}
```

## 📌 6. Endpoint dos gráficos e mapas

Para buscar os assets (gráfico + mapa) de um município:

```
GET http://localhost:3000/api/graficos/NOME_DO_MUNICIPIO/assets
```

### 🔹 Exemplo:

```
http://localhost:3000/api/graficos/Divinopolis/assets
```

## ⚠️ IMPORTANTE

* Os nomes dos municípios precisam ser válidos
* O backend normaliza automaticamente (acento, espaço, etc.)
* Os arquivos estão armazenados no Supabase Storage
